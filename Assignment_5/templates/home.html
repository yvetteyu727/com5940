Front-End

This website is built on Bootstrap with data visualization support provided by jQuery dataTable, C3js and Leaflet.

To login, just enter "demo@demo.com" for email and "password" for password.

Web App Framework

The app server is built on the Python Flask framework with the jinja template system providing the needed views for building the front-end user interface. 

Database

The Airtable database web service is accessed through the Airtable REST API call and the MySQL database is accessed through the Flask SQLAlchemy ORM library to provide data objects mapping, pagination and CRUD operaton support.

Airtable Data Source

The data used in the dashboard demonstration can be found in this Airtable database. The chart receives its data from the "Roll-up" table and the map from the "Venues" table. The album data is stored in MySQL datase. You can try add, update, or delete products from the album by using the add, update, and delete buttons above the display grid.

The dataset was originally derived from a tutorial by d3-media on learning Refine for data cleaning and transformation. The original link is here.
